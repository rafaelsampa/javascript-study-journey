## Instalar:
Node.js

## Double check no terminal se tem o node e npm:
```ssh
node -v
npm -v
```

## YARN que é um gerenciador de dependências, semelhante ao NPM porém mais rápido

```ssh
npm install -g yarn 
npm i -g pnpm
````
